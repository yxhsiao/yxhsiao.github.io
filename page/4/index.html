border

边框属性：对块的边框进行设置。border属性值有：
属性值 描述
border-width 边框的宽度,默认值为“midium”,用宽度值表示，如：5px
border-style 边框线的样式。默认为“none”。
dotted:虚线
dashed:点划线
solid:实线
double:双线
groove:3D沟槽边框。效果取决于边框颜色值
ridge:3D脊状边框。效果取决于边框颜色值
inset:3D嵌入边框。效果取决于边框颜色值
outset:3D起始边框。效果取决于边框颜色值
border-color 边框线的颜色。默认为文本的颜色

可以一次设定四条边的三个属性值。如果四个边线不尽相同，则需要分别设置。border-left、border-right、border-top、border-bottom

border-radius

边框圆角属性：可以一次指定四个角的圆角值，如果需要可以分别指定。 1. border-top-left-radius 2. border-top-right-radius 3. border-bottom-left-radius 4. border-bottom-right-radius
Value Description
length Defines the shape of the corners. Default value is 0.
% Defines the shape of the corners in %

background背景属性

CSS背景属性用于定义元素的背景效果

  • background-color 背景颜色
  • background-image 图片背景 默认定位在left-top开始
    • background-image:url()
  • background-repeat 默认图片水平、垂直平铺，可以通过此属性定义重复特点
    • background-repeat: none
    • background-repeat: repeat-x 水平平铺
    • background-repeat: repeat-y 垂直平铺
    • background-repeat: no-repeat 不重复
  • background-attachment 背景图片在no-repeat下设置图片是固定的还是滚动的
    • background-attachment: fixed 固定
    • background-attachment: scroll 滚动
  • background-position
    • left-top
    • right-top
    • left-bottom
    • right-bottom
    • center

background-image:url不指定其他属性值()

Copy复制命令

发表于 2020-03-03 Bearbeitet am 2020-03-06 分类于 计算机应用 ， DOS

DOS命令岁随着DOS的远去，大家都基本记不起他了。甚至很多天天捧着电脑办公的人连DOS是什么都不知道了…..但是就在当下的WIN10年代，系统中依然保持有CMD命令终端可以使用。你会快速的批量复制、移动文件？也许你会说：“快捷键啊，快速的CTRL+C 和CTRL+V”,如果只是这样，那么这里会教你另一种高效的方法。

单个文件的复制

copy+ 【源文件路径】+ 【文件名】+【目标路径】+ 【新文件名】

目标路径缺省时则为当前目录,新文件名缺省时则不重新命名。

举例：单文件复制

  1. 复制C盘目录下的123.txt文件到D盘目录下。
  1. 复制C盘目录下的123.txt文件到D盘目录下，并更名为abc.bat
  1. 将当前目录下的123子目录中的文件全部复制到当前目录。

举例：批量复制文件

  • 复制c:\ 123 目录中所有文件到 d:\abc 目录中
  • 当目标路径下有同名文件存在时，会出现提示“是否覆盖、不覆盖、全部覆盖” /y 参数即可实现重名文件无须确认直接覆盖。
  • 复制桌面上的所有PNG图片文件存档到E盘。将要复制某类文件用通配符*+后缀名

合并文件

  1. 将文件1和文件2合并并将新文件命名为N

  2. 将d盘根目录下的1.mp3和2.mp3合并复制到e盘下成为3.mp3

  1. 只有两个文件名，没有指定新文件名。默认为合并后文件名是第一个文件的文件名。这里就是合并为1.mp3 。mp3文件合并后可以实现连续播放。
