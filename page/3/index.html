font

字体属性：设置字体的样式、粗细、大小、字体家族等。如果格式需要将一段文字设成想要的样子，我的样子变成这样了哦！和我前后的兄弟长的怎么不一样了呢？这就需要通过font属性来指定其相应的属性值。

具体有这些属性：

  1. font-style:字体样式

    1. normal–如果标准样式nomal可以省略
    2. italic–斜体
    3. oblique–与italic相似但有区别
    4. inherit–继承

  2. font-weight:字体粗细。也可以用100～900多个等级来指定

    1. normal
    2. bold
    3. bolder
    4. lighter

  3. font-size:字体大小。数字+单位。单位有：

    1. pt

    2. px

    3. pc

    4. em

    5. rem

  4. font-family:字体系列，例如：

    1. 华文仿宋、
    2. 华文中宋、
    3. 华文楷体、
    4. 华文行楷、
    5. 微软雅黑、
    6. 等线、等线light)

可以用font一次指定所有属性或多个属性，也可以分别用font-style, font-weight, font-size, font-family来个别指定其属性值。

float

float属性指定元素应如何浮动。float属性用于定位和格式化内容，例如让图像向左浮动到容器中的文本。块级元素指定了float属性后就像一个气球一样向上飘浮。
块元素本身具有独占一行的特性。但是可以被浮动元素往一边挤开。被浮动的元素下部的元素也会

  • float:left
  • float:right

columns ：

列属性：将页面分为多列，对于新闻报纸比较适用。

  • column-width 列宽 省略时按照指定的列数自动平均分配
  • column-count 列数 将页面分为多列
  • column-gap 列间距
  • column-rule 列界线的宽度、样式，颜色

column-width将定义每列的最小宽度，而column-count定义最大列数。 通过使用此属性，多列布局将自动以窄的浏览器宽度分解为单列，而无需媒体查询或其他规则。
