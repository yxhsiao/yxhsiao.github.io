width/height

宽、高属性： 对于块级元素来说就像个盒子，那么盒子的尺寸宽和高是可以设定的。没有设定时，盒子会根据内部存放内容的大小自动调整。如果设定了宽度，那么内容超过后会自动换行。如果设定高度后，宽度自动到最大或设定值都不能容纳内部内容则内容会溢出。 1. width (500px;50%) 2. height (300px;20%) 因为终端浏览器视口大小不一致，有些时候建议还是设置成比例比较好。 举例：
