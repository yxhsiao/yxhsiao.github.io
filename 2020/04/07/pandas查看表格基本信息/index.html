pandas查看表格基本信息

发表于 2020-04-07 分类于 计算机应用 ， python Aufrufe:

[toc]

读取进来的表格或者自己新建的表格具体长什么样？怎样快速的了解DataFrame的基本信息？**

格式查看 df.info()
  1. 表格的行数
  2. 表格的列数
  3. 各列数据的类型
  4. 占用内存大小
查看列名称 df.columns
概览统计信息 df.describe()

自动统计表内数值列的信息。如：记数、平均数、方差、最小、最大、等

查看前几行或末几行的数据：df.head() df.tail()
查看表格数据 df.values

打印出来是个列表，等效于 print(df.values.tolist())

查看表格的维度 df.shape

输出表格的行数和列数。这个信息在df.info()中已经展示出来了的。

查看表格的空值 df.isnull()

DataFrame中空值默认是用“NAN”表示的。

也能单独查看某列的空值情况。

以上这样总会展示全部表格或者整列，需要你去逐个浏览，数据多时一样难以发现。但是可以结合.any()就简单明了。

.isnull().any() 判断结果用bool值输出。分别判断每列中是否有空值

查看列的唯一值 .unique()
统计列中各值的个数 .value_counts()

有如下可选参数配合使用：

  1. sort=True： 是否要进行排序；默认进行排序
  2. ascending=False： 默认降序排列；
  3. normalize=False： 是否要对计算结果进行标准化并显示标准化后的结果，默认是False。
  4. bins=None： 可以自定义分组区间，默认是否；
  5. dropna=True：是否删除缺失值nan，默认删除
