快速在当前目录下打开CMD命令行

  2. 输入CMD回车

STEP1.2 生成密钥文件并验证

  • 找到生成的密钥文件：C:\Users\yxhsi\.ssh\id_rsa.pub用记事本打开并将全部内容复制。
  • 打开GitHub_Settings_keys 页面，新建new SSH Key
