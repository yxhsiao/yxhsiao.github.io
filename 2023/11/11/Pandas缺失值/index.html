    删除缺失值

    1. df.dropna(axis,how,threst,subset,inplace)
    2. axis:0为行 1为列，default 0，数据删除维度
    3. how:{‘any’, ‘all’}, default ‘any’，any：删除带有nan的行；all：删除全为nan的行
    4. threst:int，保留至少 int 个非nan行
    5. subset:list，在特定列缺失值处理
    6. inplace:bool，是否修改源文件
