认识 list

  • list 是一个序列
  • 存在于中括号内
  • list 中元素的类型可以多样，同一个 list 中的元素的类型可以是字符串（str）、整型（int）、布尔型（Boolean）、以及嵌套的 list 等

list 的索引

下标索引

  • 注意索引是从 0 开始，最后一个的索引编号为 n-1，即所有元素的编号依次为(0,1, 2, …, n-1)。

切片

  • list[start: end : step]
  • 包含 “start”，但不包含 “end”
  • “step” 表示步长，默认情况下，“step”为 1

用for 循环遍历列表

列表的增、改、删

  • list 是一个可变的序列，可以通过添加、修改、删除等操作来操作 list 中的元素。

append() 方法

  • 是在 list 的末尾添加元素
    ['北京', '上海', '天津', '重庆', '深圳']

insert()

  • 是在指定位置添加元素。
    ['北京', '广州', '上海', '天津', '重庆', '深圳']

用 pop()

  • 方法删除 list 末尾的元素

用 pop(i) 方法

  • 删除指定位置的元素

修改 list 中的元素

  • 直接通过 list 的索引进行赋值来实现

‘ + ’ 拼接列表

‘ * ‘ 重复列表

  • 返回列表元素最小值

  • 将其他序列转成列表

list的其他方法

list.count(obj)

  • 统计某个元素在列表中出现的次数

list.extend(seq)

  • 在列表末尾一次性追加另一个序列中的多个值（用新列表扩展原来的列表）

list.index(obj)

  • 从列表中找出某个值第一个匹配项的索引位置

list.remove(obj)

  • 移除列表中某个值的第一个匹配项
    ['u', 'a', 'n', 'g', 'z', 'h', 'o', 'u']

list.sort()

  • 对原列表进行排序

list.reverse()

  • 对原列表进行反向排序

列表推导式 (解析式)

一般语法结构：

  • new_list = [x for x in iterable]
  • 其中的 iterable 表示可迭代的对象，包括字符串（str）、列表（list），元组（tuple）、字典（dict）、集合（set）以及生成器（generator）等。

两层 for 循环的列表推导式

使用两个变量来生成 list

  • 列表推导式也可以使用两个或多个变量来生成 list，结合字典的使用

列表的去重

  1. 直接用set方法可以去重，但是原列表中元素的顺序被打乱
  2. 利用原来各元素在列表中的索引位可以保持原来的顺序
